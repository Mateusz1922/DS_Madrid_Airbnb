Madrid Airbnb Data Science Project

dataset: https://www.kaggle.com/datasets/rusiano/madrid-airbnb-data/data

Implemented with Python

The project included:
- Data extraction and preprocessing
- Dataset description and analysis
- Data visualization
- Performing data prediction and relationship discovery using different
  machine learning techniques like Linear Regression, MLP, Decision Trees
  Gradient Boosting trees, random forest
- Final conclusions and visualization of he results
- Report explaining our approach and results
